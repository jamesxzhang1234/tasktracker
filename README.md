# tasktracker

Download node, npm, ts-node, and ts

- run "ts-node index.ts [insert command here]" in command prompt/terminal

Commands :

Perform command then add the following () 

add (string) 
update (id) (string)
delete (id)
clear 
mark-done (id)
mark-in-progress (id)
list (todo, in-progress, done)
